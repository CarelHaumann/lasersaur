Tips

  • Build on a planar surface.
  • The "preassembly" T-slot nuts can be side-loaded into the slot. Pre-loading is not necessary.
  • Aim for tolerances of less than 0.5mm (0.02") basically making sure there are no gaps or overhanging edges.
  • Use left and right arrow key to navigate this guide.
  • Use shift-left and shift-right arrow key to flip through vantage point carousel.
  • Main image resizes with browser window. Make more narrow to fit more vertically.
